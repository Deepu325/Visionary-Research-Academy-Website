# Visionary Research Academy

A professional, conversion-focused website for academic research guidance and mentoring services.

## Project Overview

This is a modern React-based website designed for Visionary Research Academy, providing ethical academic research guidance for undergraduate, postgraduate, and PhD scholars.

### Key Features

- **Trust-focused design** with academic credibility
- **Conversion-optimized** lead generation
- **Responsive** mobile-first design
- **Accessible** WCAG AA compliant
- **SEO-friendly** semantic HTML structure

### Pages Structure

1. **Home** - Hero section with clear value proposition
2. **Services** - Detailed service offerings by academic level
3. **Pricing** - Transparent pricing tiers and packages
4. **Success Stories** - Testimonials and case studies
5. **FAQ** - Comprehensive questions and answers
6. **Contact** - Lead capture forms and consultation booking

### Design System

**Colors:**
- Primary: #003366 (Navy Blue - trust & authority)
- Secondary: #28A745 (Forest Green - growth & ethics)
- Accent: #FFD700 (Gold - optimism, used sparingly)
- Background: #F8F9FA
- Text: #212529

**Typography:**
- Font: Inter (Google Fonts)
- Hierarchy: Clear section titles and readable body text

### Tech Stack

- React 18 with Vite
- React Router for navigation
- CSS-in-JS with inline styles
- Responsive grid layouts
- Modern ES6+ JavaScript

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
